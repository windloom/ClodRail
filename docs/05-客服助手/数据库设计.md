# 客服助手数据库设计

数据库：`rs-assistant`

表结构以 Docker MySQL 容器中的实际 DDL 为准。

## 对话消息表 (`chat_message`)

```sql
CREATE TABLE `chat_message` (
  `id` int NOT NULL AUTO_INCREMENT COMMENT '主键',
  `session_id` varchar(50) NOT NULL COMMENT '对话id',
  `content` text NOT NULL COMMENT '对话',
  PRIMARY KEY (`id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci COMMENT='对话id内容表';
```

## 会话记忆表 (`memory`)

```sql
CREATE TABLE `memory` (
  `id` int NOT NULL AUTO_INCREMENT COMMENT '主键',
  `session_id` varchar(50) NOT NULL COMMENT '对话id',
  `content` text NOT NULL COMMENT '历史 ',
  `type` int DEFAULT NULL COMMENT '对话类型 1- AI对话 2- 人工客服',
  `user_id` int DEFAULT NULL,
  `agent_id` int DEFAULT NULL,
  `update_time` datetime DEFAULT NULL,
  `create_time` datetime DEFAULT NULL,
  PRIMARY KEY (`id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci COMMENT='对话id内容表';
```
